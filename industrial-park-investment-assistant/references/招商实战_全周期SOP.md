# 招商实战 · 全周期SOP

> **使用说明：** 本文件为从首次接触到达成签约/入驻的全周期标准流程和时间节点。
> 安装后可按本园区的实际流程调整时间节点和检查项。

---

## 招商全周期流程图

```
首次接触 ──→ 需求确认 ──→ 方案报价 ──→ 带看洽谈 ──→ 谈判签约 ──→ 入驻服务
    │            │            │            │            │            │
  Day 1        Day 3        Day 7       Day 14       Day 30       Day 60+
```

---

## 阶段一：首次接触（Day 1）

**目标：** 留下专业印象，获取初步需求信息

**标准动作：**
- [ ] 确认客户来源渠道（主动咨询/转介绍/活动获客/扫楼等）
- [ ] 记录客户基础信息（企业名/联系人/联系方式）
- [ ] 了解初步需求（面积/类型/预算/选址时间）
- [ ] 调用Agent查询匹配房源并反馈
- [ ] 约定下次跟进时间或带看安排

**Agent辅助：**
- 输入：「客户是XX公司，行业XX，需求XX」
- Agent输出：客户画像初稿 + 匹配房源 + 初步沟通建议

**时效要求：** 首次响应不超过4小时（工作时间）

---

## 阶段二：需求确认（Day 1-3）

**目标：** 深度摸清需求，确认方向匹配

**标准动作：**
- [ ] 深入了解企业背景（行业/规模/融资/分支机构）
- [ ] 确认产品类型需求（研发/生产/办公/总部）
- [ ] 摸清面积、层高、承重、电力等硬性参数
- [ ] 了解选址时间表和决策链
- [ ] 确认预算范围和价格敏感度
- [ ] 评估产业匹配度
- [ ] 完成客户画像定稿

**Agent辅助：**
- 输入：「帮我完善客户画像，已知XX/XX/XX」
- Agent输出：完整客户画像 + 产业匹配分析 + 风险提示

**输出物：** 客户画像表（存入知识库/CRM）

---

## 阶段三：方案报价（Day 3-7）

**目标：** 给出有竞争力的方案，推动带看

**标准动作：**
- [ ] 根据需求匹配2-3套备选方案
- [ ] 整理租金报价（含优惠机制）
- [ ] 梳理客户可享受的政策清单
- [ ] 制作方案对比表
- [ ] 主动发送方案并跟进确认

**Agent辅助：**
- 输入：「XX公司，需求800㎡研发，预算3块以内，帮我出方案」
- Agent输出：2-3套方案含报价 + 政策清单 + 方案话术

**输出物：** 定制化招商方案（含房源/租金/政策/入驻时间表）

**时效要求：** 方案出具不超过48小时

---

## 阶段四：带看洽谈（Day 7-14）

**目标：** 通过实地带看，推动意向升级

**带看前准备（Agent辅助）：**
- [ ] 根据行业类型确认带看重点
- [ ] 准备园区资料包（含实景照片/视频/VR）
- [ ] 预约带看路线和讲解人员
- [ ] 确认客户决策链成员是否到场

**带看中：**
- [ ] 按标准流程讲解（园区概况→产品→配套→政策）
- [ ] 针对客户行业重点展示（如AI企业重点展示网络/电力）
- [ ] 记录客户现场反馈和疑虑
- [ ] 结束时约定下一步

**带看后（Day 1内）：**
- [ ] 发送带看总结邮件
- [ ] 补充客户现场提出的问题和回答
- [ ] 跟进客户反馈

**Agent辅助：**
- 输入：「XX公司明天带看，行业是XX，重点看XX」
- Agent输出：带看清单 + 讲解话术 + 现场可能问题及应答

---

## 阶段五：商务谈判（Day 14-30）

**目标：** 达成签约意向

**谈判前：**
- [ ] 明确谈判目标和底线（租金/免租期/递增率等）
- [ ] 准备好优惠政策包（灵活使用的筹码）
- [ ] 了解竞品当前报价策略
- [ ] 确认园区领导授权范围

**谈判中：**
- [ ] 先谈价值再谈价格
- [ ] 每次让步都要有条件交换
- [ ] 注意节奏，不要一次让步太多
- [ ] 及时记录客户诉求

**谈判后：**
- [ ] 当天出谈判备忘录
- [ ] 确认合同主要条款
- [ ] 约定合同审核时间节点

**Agent辅助：**
- 输入：「进入谈判阶段，客户提供XX诉求，我方底线是XX」
- Agent输出：谈判策略 + 让步空间清单 + 话术建议

---

## 阶段六：签约入驻（Day 30-60+）

**签约前：**
- [ ] 确认合同条款（面积/租金/递增/免租/物业条款）
- [ ] 法务/财务审核
- [ ] 双方确认合同文本
- [ ] 安排签约仪式（如需要）

**签约：**
- [ ] 收取定金/履约保证金
- [ ] 完成合同签署
- [ ] 发送签约确认和入驻须知

**入驻准备：**
- [ ] 协助客户办理装修/改造手续
- [ ] 跟进装修设计和施工
- [ ] 协调入驻时间、钥匙交接
- [ ] 介绍物业和配套设施
- [ ] 建立客户档案和续租跟踪

**Agent辅助：**
- 输入：「XX公司签约了，接下来入驻要准备什么？」
- Agent输出：入驻准备清单 + 物业对接事项 + 客户档案模板

## 关键时间节点检查清单

| 天数 | 检查项 | 完成标志 |
|------|--------|---------|
| Day 1 | 完成首次接触并录入系统 | 客户画像草稿已创建 |
| Day 3 | 完成需求确认 | 客户画像定稿 |
| Day 7 | 完成方案报价 | 方案已发送客户 |
| Day 14 | 完成首次带看 | 带看总结已发送 |
| Day 30 | 进入签约谈判或明确意向状态 | 进入签约流程或客户决策 |
| Day 45 | 签约或明确放弃 | 签约完成或客户转竞品 |
| Day 60 | 完成入驻 | 钥匙交接完成 |

## 超期预警规则

**Agent应在以下情况下发出预警：**

| 预警等级 | 触发条件 | 建议动作 |
|---------|---------|---------|
| 🔴 红 | A类客户连续7天无跟进 | 立即电话/上门联系 |
| 🟡 黄 | 任何客户连续14天无跟进 | 发送园区动态激活联系 |
| 🔵 蓝 | 进入签约阶段超过30天未签 | 了解障碍，对症下药 |
| ⚪ 灰 | B/C类客户连续30天无跟进 | 降级处理，保留记录 |
