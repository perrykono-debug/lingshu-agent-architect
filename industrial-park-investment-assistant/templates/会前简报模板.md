## 三、对话剧本（针对不同决策人）

### 3.1 如果决策人是 CEO/创始人

**切入点：** 谈战略、谈未来、谈产业生态

**破冰话题：**
- 「 Congratulations on your Series B funding! 我看了你们的最新专利，在 [X] 领域很有突破性…」
- 「你们最近招聘的 [X] 岗位，是不是业务扩张很快？」

**关键问题：**
1. 「未来3年的战略重心是什么？人才扩招计划如何？」
2. 「现有场地有哪些痛点？是否满足研发/测试需求？」
3. 「选址最看重哪些因素？成本、配套、还是产业生态？」

**价值传递：**
- 园区产业生态：[X] 家同产业链企业，协同效应强
- 人才供给优势：周边 [X] 高校，年输送毕业生 [X] 人
- 政策支持：[X] 万元/年的补贴，3年可节省 [X] 万元

**可能的异议 + 应对：**
- ❓ 「租金太贵」 → 「我们算一下TCO，3年综合成本其实比 [竞品] 低 [X]%，而且我们的政策补贴可以抵扣 [X] 元/年…」
- ❓ 「位置有点远」 → 「我们提供 [X] 班车路线，到地铁站 [X] 分钟，而且周边配套完善，员工生活成本低…」

### 3.2 如果决策人是 COO/运营负责人

**切入点：** 谈成本、谈效率、谈配套服务

**破冰话题：**
- 「你们现在的场地，听说交付标准不太满足研发需求？」
- 「我看了你们的团队规模，扩张很快啊，现有场地还够用吗？」

**关键问题：**
1. 「现有场地的交付标准有哪些不满足的地方？」
2. 「计算过搬迁的时间成本和装修成本吗？」
3. 「对园区的配套服务（食堂/公寓/会议）有哪些要求？」

**价值传递：**
- 精装交付：即租即用，0装修成本，0时间成本
- 配套完善：食堂/人才公寓/会议中心/路演厅，降低企业运营成本 [X]%
- TCO优势：3年综合成本比毛坯交付园区低 [X]%

**可能的异议 + 应对：**
- ❓ 「你们物业费太高」 → 「物业费包含 [X] 项服务（安保/清洁/绿化/设施维护），折算下来实际 [X] 元/㎡/月，而且…」
- ❓ 「精装交付的装修标准如何？」 → 「我们提供3种装修标准，可以满足研发/办公/展示等不同需求，而且…」

### 3.3 如果决策人是 CFO/财务负责人

**切入点：** 谈TCO、谈政策、谈税收优惠

**破冰话题：**
- 「你们财务测算过搬迁的成本吗？除了租金，还有哪些隐形成本？」
- 「我们园区有专门的财务咨询服务平台，可以帮企业做税务筹划…」

**关键问题：**
1. 「除了租金和物业费，企业选址还会考虑哪些成本？」
2. 「了解过上海市和宝山区的产业政策吗？补贴力度很大…」
3. 「对园区提供的金融服务（产业基金/融资对接）有兴趣吗？」

**价值传递：**
- TCO透明：3年综合成本 [X] 万元，竞品 [X] 万元，节省 [X] 万元
- 政策收益：[X] 万元/年的补贴（人才+税收+专项），直接抵扣租金
- 金融服务：园区产业基金 [X] 亿元，可跟投优质企业 [X] 万元

**可能的异议 + 应对：**
- ❓ 「我们需要详细的成本测算对比」 → 「我这就给您出一份TCO成本对比报告，包含租金、物业费、停车费、装修分摊、政策收益、时间成本，3年综合对比…」
- ❓ 「政策补贴能否兑现？」 → 「这是宝山区和中集产城的官方协议，已经有很多企业成功申报，补贴按时到账…」

### 3.4 如果决策人是 CTO/技术负责人

**切入点：** 谈层高、谈承重、谈电力、谈网络

**破冰话题：**
- 「你们研发实验室对层高和承重有什么要求？我们园区有 [X] 米层高、[X] kg/㎡承重的房源…」
- 「听说你们需要做 [X] 测试，对电力负荷要求很高？我们园区可以做到 [X] KW/层…」

**关键问题：**
1. 「研发实验室/测试中心对场地有哪些特殊要求？」
2. 「现有场地的电力、网络、通风是否满足需求？」
3. 「对园区的技术服务（算力平台/测试平台）有需求吗？」

**价值传递：**
- 硬件优势：层高 [X] m、承重 [X] kg/㎡、电力 [X] KW/层，满足研发需求
- 技术服务：人工智能算力平台、公共测试平台，降低企业研发成本 [X]%
- 智慧园区：5G全覆盖、物联网平台，提升研发效率 [X]%

**可能的异议 + 应对：**
- ❓ 「我们需要 [X] KW的电力增容，能做到吗？」 → 「园区电力冗余充足，增容申请 [X] 天内可批复，而且…」
- ❓ 「网络带宽和稳定性如何？」 → 「园区接入 [X] 条光纤，带宽可达 [X] Gbps，而且有备用线路，保证99.9%可用性…」
