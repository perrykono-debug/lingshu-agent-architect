# 对话剧本模板

> 本文档为AI生成针对不同决策人的对话剧本提供结构化模板。
> AI根据企业实际情况和决策人类型，自动选择合适的剧本并填充具体内容。

---

## 使用说明

**触发词：**「对话剧本」「见XX总怎么聊」「帮我准备话术」「XX总关注什么」

**AI执行步骤：**
1. 识别决策人类型（CEO/COO/CFO/CTO/其他）
2. 读取企业背景和扩张信号
3. 选择合适的剧本模板
4. 填充所有 `[X]` 占位符（企业名/决策人姓名/具体数据）
5. 输出可直接使用的对话剧本

## 剧本1：面对 CEO/创始人

**适用场景：** 企业最高决策者，关注战略、未来、产业生态

### 破冰阶段（前3分钟）

**目标：** 建立信任，找到共同点

**话术示例：**
```
「[X总您好]，我是[园区名称]的[你的姓名]。
    之前关注到贵司在[领域X]的最新突破，特别是[具体技术/产品X]，
    在[应用场景X]很有优势。[X总]当时创业，是看到什么机会？」
```

**关键点：**
- 展示做过功课（了解企业最新动态）
- 用提问引导对方分享（人喜欢聊自己的成功）
- 避免一上来就推销

### 需求挖掘阶段（3-10分钟）

**目标：** 了解扩张计划、痛点、决策标准

**关键问题清单：**
1. 「未来3年的战略重心是什么？团队扩招计划如何？」
2. 「现有场地有哪些痛点？是否满足[研发/测试/办公]需求？」
3. 「选址最看重哪些因素？成本、配套、还是产业生态？」
4. 「决策周期大概多长？关键决策节点有哪些？」

**倾听要点：**
- 记关键词（扩招/搬迁/成本/人才/政策）
- 识别隐性需求（不只是面积，还有产业生态、人才供给）
- 判断决策紧迫度（Q3前必须完成 vs 看看再说）

### 价值传递阶段（10-20分钟）

**目标：** 将园区优势与客户需求匹配

**价值传递框架：**

**A. 产业生态优势**
```
「[园区名称] 聚焦 [主导产业X]，已入驻 [X] 家产业链企业，
    包括 [链主企业X]、[核心供应商X]。
    贵司做 [业务X]，正好可以和 [园区企业X] 形成协同，
    比如 [具体协同场景X]...」
```

**B. 人才供给优势**
```
「周边有 [X] 所高校，年输送 [专业X] 毕业生 [X] 人。
    园区还有人才公寓，租金比市中心低 [X]%，
    贵司 [X] 人的团队，一年能省 [X] 万住宿成本...」
```

**C. 成本优势（TCO视角）**
```
「我们算过一笔账，贵司 [X]㎡，3年TCO（总拥有成本）
    我们比 [竞品X] 低 [X]%，
    而且我们有 [X] 万/年的政策补贴，可以直接抵扣租金...」
```

**D. 时间优势（精装交付）**
```
「我们是精装交付，即租即用，0装修成本，0时间成本。
    如果选毛坯，装修 [X] 个月，按贵司月营收 [X] 万计算，
    相当于损失 [X] 万营收...」
```

### 异议处理阶段（20-30分钟）

**常见异议 + 应对话术：**

**异议1：「租金太贵了」**
```
「[X总]，租金只是显性成本，我们算一下TCO（总拥有成本）：
    1. 租金：[X]元/年
    2. 物业费：[X]元/年（含安保/清洁/设施维护）
    3. 停车费：[X]元/年（[X]个车位）
    4. 装修分摊：[X]元/年（精装0元）
    5. 政策收益：- [X]元/年（人才补贴+税收返还）
    ------------------------
    实际年净成本：[X]元
    
    竞品 [竞品X] 表面租金低 [X]%，但：
    - 毛坯交付，装修 [X] 万（分摊3年 = [X]元/年）
    - 无政策补贴，多支出 [X]元/年
    - 配套不足，员工餐饮/会议需外出，效率低 [X]%
    
    实际年净成本：[X]元
    
    我们3年实际节省：[X]元，相当于多招 [X] 个核心工程师...」
```

**异议2：「位置有点远」**
```
「[X总]，位置确实是个考虑因素，但我们提供 [X] 条班车路线，
    到地铁站仅 [X] 分钟。
    
    而且，您考虑过员工的生活成本吗？
    - 周边人才公寓：[X]元/月/人（市中心 [X]元/月/人）
    - 餐饮成本：园区食堂 [X]元/餐（外面 [X]元/餐）
    - 时间成本：周边生活配套完善，员工满意度高 [X]%
    
    综合算下来，员工实际收入反而更高，留存率提升 [X]%...」
```

**异议3：「我们再对比对比」**
```
「完全理解，选址是重要决策，应该多对比。
    我帮您梳理一下对比维度，避免遗漏：
    
    1. TCO（总拥有成本）：租金+物业+停车+装修-政策收益
    2. 时间成本：装修周期/搬迁周期/正式运营时间
    3. 产业生态：上下游协同/产业链对接机会
    4. 人才供给：高校资源/人才公寓/生活成本
    5. 政策支持：补贴力度/兑现难度/长期稳定性
    
    我给您出一份《3年TCO成本对比报告》，
    包含我们和 [竞品X]、[竞品Y] 的详细对比，
    您拿给团队一起评估，也欢迎随时问我...」
```

### 推进成交阶段（30分钟后）

**目标：** 明确下一步行动，推动决策

**推进话术示例：**
```
「[X总]，今天沟通下来，感觉我们园区和贵司需求匹配度很高。
    建议下一步：
    
    1. 我给您出一份正式的《TCO成本对比报告》+《政策申报指南》，
       您内部评估一下。
    2. 安排一次实地考察，带您看看 [X] 楼 [X]㎡ 的精装房源，
       也见见已入驻的 [产业链企业X]，聊聊实际体验。
    3. [日期X] 前，如果您有兴趣，我们可以谈具体租赁方案和优惠。
    
    您看 [日期X] 方便吗？我提前安排...」
```

**Closing技巧：**
- 给选择题，不是问答题：「您看周三还是周四方便？」
- 制造紧迫感（真实）：「6月30日前签约，有 [X]% 折扣」
- 降低决策风险：「可以先签意向书，锁定房源和优惠，7天内可退」

## 剧本2：面对 COO/运营负责人

**适用场景：** 关注成本、效率、配套服务

### 破冰阶段

**话术示例：**
```
「[X总您好]，我是[园区名称]的[你的姓名]。
    听说贵司最近业务扩张很快，团队要从 [X] 人扩到 [X] 人？
    [X总]作为COO，场地这块肯定压力不小...」
```

**关键点：**
- 直接切入业务痛点（扩张/成本/效率）
- 展示共情（理解COO的压力）
- 快速建立专业形象

### 需求挖掘阶段

**关键问题清单：**
1. 「现有场地的交付标准有哪些不满足的地方？」
2. 「计算过搬迁的时间成本和装修成本吗？」
3. 「对园区的配套服务（食堂/公寓/会议）有哪些要求？」
4. 「团队日常运营中，最头疼的3件事是什么？」

### 价值传递阶段

**A. 成本优势（TCO视角）**
```
「我们精装交付，即租即用，0装修成本，0时间成本。
    如果选毛坯，装修 [X] 个月，成本 [X] 万，
    而且期间团队效率下降 [X]%，相当于损失 [X] 万...」
```

**B. 配套优势（降低企业运营成本）**
```
「园区配套非常完善：
    - 食堂：[X]个食堂，[X]元/餐，满足 [X] 人同时就餐
    - 人才公寓：[X] 套，[X]元/月/间，步行 [X] 分钟
    - 会议中心：[X] 个会议室，最大 [X] 人，免费预约
    - 路演厅：[X] 个，配备 [X] 设备，适合产品发布/行业峰会
    
    贵司 [X] 人团队，一年能省 [X] 万运营成本...」
```

**C. 服务优势（C+运营服务体系）**
```
「我们有C+运营服务体系，不只是物业：
    - 企业服务：政策申报/财税代理/法律咨询/人力资源
    - 产业服务：产业链对接/技术平台共享/市场资源对接
    - 生活服务：食堂/公寓/健身/社交活动
    
    [X] 家企业实践证明，入驻后企业运营成本平均降低 [X]%...」
```

### 异议处理阶段

**常见异议 + 应对话术：**

**异议1：「物业费太高了」**
```
「[X总]，物业费确实是个成本，但我们要看包含什么：
    1. 安保服务：[X] 人/天，[X] 小时巡逻
    2. 清洁服务：[X] 次/天，公共区域+卫生间
    3. 绿化养护：[X] 次/月，办公环境舒适
    4. 设施维护：[X] 分钟内响应，[X] 小时内解决
    5. 客户服务：[X] 小时在线，响应速度 [X] 分钟
    
    如果外包这些服务，一年至少 [X] 万，
    我们物业费折算下来，实际 [X] 元/㎡/月...」
```

**异议2：「精装交付的装修标准如何？」**
```
「我们提供3种装修标准，可以根据贵司需求定制：
    
    标准A（基础办公）：[X] 元/㎡，包含 [X] [X] [X]
    标准B（研发办公）：[X] 元/㎡，包含 [X] [X] [X]
    标准C（展示+办公）：[X] 元/㎡，包含 [X] [X] [X]
    
    而且，我们装修材料都是 [环保标准X]，[X] 天即可入驻，
    不会像传统装修那样有甲醛问题...」
```

## 剧本3：面对 CFO/财务负责人

**适用场景：** 关注TCO、政策、税收优惠

### 破冰阶段

**话术示例：**
```
「[X总您好]，我是[园区名称]的[你的姓名]。
    听说贵司最近完成了 [X] 轮融资 [X] 亿元，
    作为CFO，选址这块肯定要精打细算...」
```

**关键点：**
- 展示对财务数据的关注（融资/成本/ROI）
- 用财务语言沟通（TCO/ROI/现金流）
- 快速建立信任（专业/靠谱/为企着想）

### 需求挖掘阶段

**关键问题清单：**
1. 「除了租金和物业费，企业选址还会考虑哪些成本？」
2. 「了解过上海市和宝山区的产业政策吗？补贴力度很大...」
3. 「对园区提供的金融服务（产业基金/融资对接）有兴趣吗？」
4. 「选址的预算上限是多少？决策时最看重ROI还是现金流？」

### 价值传递阶段

**A. TCO优势（透明/全面/可验证）**
```
「我们提供《TCO成本对比报告》，所有成本透明：
    
    直接成本：
    - 租金：[X]元/年
    - 物业费：[X]元/年
    - 停车费：[X]元/年
    - 装修分摊：[X]元/年（精装0元）
    
    政策收益：
    - 人才补贴：[X]元/年（[X]人 × [X]元/人/月 × 12）
    - 税收返还：[X]元/年（[X]%返还）
    - 专项补贴：[X]元/年（主导产业补贴）
    
    净成本：[X]元/年
    
    竞品 [竞品X] 表面租金低，但3年TCO高 [X]%...」
```

**B. 政策优势（可兑现/力度大/长期稳定）**
```
「宝山区 + 上海市 双重政策红利：
    
    1. 人才补贴：[X]元/人/月，[X] 人一年 [X] 万
    2. 税收返还：增值税+所得税 [X]% 返还，一年 [X] 万
    3. 主导产业补贴：[X] 万（一次性）
    4. 高新技术企业补贴：[X] 万（认定后）
    
    3年政策收益：[X] 万，直接抵扣租金...」
```

**C. 金融服务优势（产业基金/融资对接）**
```
「园区有 [X] 亿产业基金，可跟投优质企业 [X] 万。
    已投企业 [X] 家，平均回报率 [X]x。
    
    还有融资对接服务，对接 [X] 家投资机构，
    [X] 家企业成功融资 [X] 亿...」
```

### 异议处理阶段

**常见异议 + 应对话术：**

**异议1：「我们需要详细的成本测算对比」**
```
「完全理解，CFO必须精打细算。
    我这就给您出一份《3年TCO成本对比报告》，
    包含：
    - 租金/物业费/停车费/装修分摊
    - 政策收益（人才补贴+税收返还+专项补贴）
    - 时间成本（装修周期+搬迁周期）
    - 竞品对比（[竞品X] [竞品Y]）
    
    所有数据都可验证，您拿给团队评估...」
```

**异议2：「政策补贴能否兑现？」**
```
「这是个关键问题，我给您看实际案例：
    
    [企业X] 2025年入驻，当年拿到人才补贴 [X] 万，
    税收返还 [X] 万，一共 [X] 万，直接抵扣下年租金。
    
    [企业Y] 2024年入驻，高新技术企业认定后，
    拿到一次性补贴 [X] 万 + 3年税收返还 [X] 万...
    
    这是宝山区和中集产城的官方协议，
    已经写入合同，100%兑现...」

**适用场景：** 关注层高、承重、电力、网络

### 破冰阶段

**话术示例：**
```
「[X总您好]，我是[园区名称]的[你的姓名]。

    平均修复时间 [X] 分钟...」

## 通用推进技巧

### 推进信号识别

**客户感兴趣的信号：**
- 问具体房源（「有 [X]㎡ 的空置吗？」）
- 问政策细节（「人才补贴怎么申报？」）
- 问时间节点（「如果现在签约，什么时候能入驻？」）
- 问合同细节（「租约最短几年？能不能提前退租？」）

**客户犹豫的信号：**
- 「我们再内部讨论一下」
- 「还要对比对比其他园区」
- 「租金有点超预算了」
